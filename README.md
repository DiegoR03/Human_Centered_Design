Gesprek met:
<br>Vandaag zijn we begonnen met het vak Human Centered Design. TIjdens de uitleg in de ochtend werd de klas in vieren verdeeld en kregen wij ieder een 'klant' waar wij voor moeten werken. De klant waar ik voor ben ingedeelt is Darice. Deze klant gaf ons twee mogelijke opdrachten:
- Een website waar zij podcasts op kan luisteren en de emoties kan waarnemen doormiddel van tekst
- Een website waar zij de emoties van films beter kan waarnemen
<br> De reden dat zij deze websites wilt hebben is omdat zij later in het leven doof is geworden.
<br><br>
Ik ben begonnen door een keuze te maken, en ik ga werken aan een podcast website. Ik ben gelijk begonnen met brainstormen en ben uiteindelijk op het idee gekomen om een transcriptie website te maken. Op deze website kan je de podcast volgen en worden de transcriptie zinnen live uitgetyped onder de audio file. Om de emoties meer terug te laten komen wil ik de website zelf van styling laten veranderen. Dit houdt dan in dat bijvoorbeeld de achtergrond kleuren veranderen, vormen tevoorschijn komen of zlefs de lettertypes zelf veranderen.
Om dit voor elkaar te krijgen in de korte tijd die we hebben moest ik beginnen met de basics. Ik ben begonnen met het uitwerken van de live transcripties. Ik heb eerst een podcast uitgekozen: "We found the most powerful object in the universe - Astrum Space". Hoewel ik de klant nog niet ken, moest ik wel alvast beginnen met een podcast zodat ik de basics uit kan werken. Daarna ben ik verder gegaan met de transcripties. Deze moest ik downloaden en converten van een text bestand naar een `.vtt` bestand. Met een `.vtt` bestand kan je 'timestamps' toevoegen en kan ik met een `<track>` element en javascript ervoor zorgen dat de innerHTML van de div de huidige tekst vertoont: <br>
